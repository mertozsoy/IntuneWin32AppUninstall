# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-01

### Added
- Initial release of AtaUninstallPack
- WPF GUI for scanning and selecting MSI-installed applications
- Automatic generation of Microsoft Intune-compatible Detection and Remediation scripts
- Support for both 32-bit and 64-bit registry scanning
- Multi-select functionality for batch generation
- Context menu with quick actions
- CSV export functionality
- Refresh button to reload application list
- Detailed logging of uninstall operations
